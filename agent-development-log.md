# Agent Development Log

Project Title: Anxiety In Lines
Student / Team: Ian
Original Life Experience: Waiting in line often creates anxiety because the queue moves unpredictably, people ahead may delay progress, and the player has little control while being confined near strangers.
Core Emotion: Anxiety caused by uncertainty, waiting, limited control, and social pressure in a queue.
Core Mechanic: Queue management under a time limit. The player advances through a line, negotiates with NPCs, holds position, chooses lines, and decides whether to persuade others to move or let others go first.
Current Game Idea: A queue-based game where slow-moving NPCs and uncertain wait times create tension. The player tries to secure a ticket before time runs out while navigating negotiation, fairness, line-cutting, and unpredictable people ahead.
Current Graph / Data Structure Summary: The game is structured around NoP, GT, and RWT. NoP is the number of people remaining ahead of the player. GT is remaining game time and decreases by `Time.deltaTime`. RWT is a random waiting time for each NPC, generated by `Random.Range(A1, A2)`. Win condition: `NoP == 0 && GT > 0`. Game over condition: `NoP > 0 && GT <= 0`. Player inputs can reduce NoP by moving forward or persuading others, or increase NoP by allowing others to go first.
AI Agent Used: Codex / ChatGPT AI development collaborator
Development Period: 2026-06-09 onward
